## Project Summary

This project cleans Zepto retail inventory data, creates revenue and inventory metrics, analyzes the data with SQL, and presents the output in an interactive Streamlit dashboard.

## Important Note

The original dataset did not include city or map fields. Synthetic city, region, warehouse, latitude, and longitude fields were added only to make the dashboard more complete for portfolio presentation.
